# Веб-додаток (+ Лаб 2): Аналіз індексів рослинності України

## Реалізовано наступні функції та графіки:

1. **Автоматичне завантаження даних (Data Pipeline)** Автоматично отримує свіжі дані VHI з сервера NOAA для всіх областей України у разі їх відсутності. Забезпечує безперебійну роботу додатка та інформує користувача про процес завантаження за допомогою візуального індикатора (Progress Bar).
2. **Інтерактивна панель керування (Sidebar Filters)** Надає користувачу можливість гнучко обирати вегетаційний індекс (VCI, TCI, VHI), цільову область, а також часові проміжки (інтервал років та тижнів) з миттєвим оновленням результатів на головному екрані.
3. **Сортування та керування станом (State Management)** Забезпечує сортування табличних даних за зростанням або спаданням із захистом від конфліктів (обробка сценарію одночасного вибору обох опцій), а також швидке повернення до початкових налаштувань за допомогою кнопки Reset.
4. **Динаміка індексу у часі (Line Chart)** Візуалізує часовий ряд обраного індексу рослинності для конкретної області у вигляді інтерактивного графіка Plotly. Дозволяє досліджувати сезонні коливання, масштабувати графік та виявляти багаторічні тренди посух.
5. **Багатовимірне порівняння областей (Bar Chart)** Відображає середні значення обраного показника по всіх регіонах України за визначений користувачем часовий інтервал. Застосовує кольорове кодування для яскравого підсвічування цільової області, що значно спрощує порівняльний аналіз.
