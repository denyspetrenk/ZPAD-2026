# Візуалізація та аналіз даних

## Датасет:

**Назва:** [Communities and Crime](https://archive.ics.uci.edu/dataset/183/communities+and+crime)

**Джерело:** UCI Machine Learning Repository (ID: 183)

Датасет містить дані про громади в США, об'єднуючи соціально-економічні показники (перепис 1990 року), дані правоохоронних органів (LEMAS 1990) та статистику злочинності (FBI UCR 1995). Усі числові дані в датасеті попередньо нормовані в діапазоні [0, 1].

## Реалізовано наступні графіки:

1. **Залежність злочинності від доходу (Scatterplot)** Візуалізує зворотний зв'язок: чим вищий медіанний дохід у районі, тим нижчий рівень насильницьких злочинів.
2. **Розподіл по 5 діапазонах (Histplot)** Показує кількість громад у п'яти рівних групах інтенсивності злочинності. Допомагає побачити, що більшість районів мають низький рівень злочинності.
3. **Багатовимірний аналіз (Bubble Chart)** Об'єднує 4 параметри: дохід, злочинність, населення (розмір точки) та безробіття (колір). Демонструє, що безробіття є сильнішим фактором ризику, ніж розмір громади.
4. **Теплова карта кореляції (Heatmap)** Відображає силу зв'язку між ключовими ознаками. Наприклад, підтверджує високу кореляцію між бідністю та рівнем злочинності (0.52).
5. **Аналіз аномалій (Boxplot)** Допомагає знайти "викиди" — райони з аномально високою злочинністю (понад 0.7), які статистично виділяються на фоні інших громад.
6. **Щільність розподілу бідності (KDE Plot)** Показує плавний розподіл показника бідності, дозволяючи зрозуміти, яка частка населення найчастіше перебуває у критичному фінансовому стані.
