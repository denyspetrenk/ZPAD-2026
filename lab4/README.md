# Візуалізація даних 2

## Реалізовано наступні функції та графіки:

1. **Чиста гармоніка (Sine Wave)** Відображає ідеальну математичну модель сигналу за базовою формулою $y(t) = A\sin(\omega t + \phi)$. Дозволяє візуально оцінити вплив амплітуди, частоти та фази на форму хвилі.
2. **Генерація шуму (Normal Distribution)** Накладає на базову гармоніку випадковий нормальний шум. Імітує реальні фізичні перешкоди або похибки датчиків із можливістю керувати середнім значенням та дисперсією.
3. **Фільтрація сигналу (Butterworth Low-pass Filter)** Застосовує цифровий фільтр низьких частот для очищення зашумленого сигналу. Використання двонаправленої фільтрації усуває фазовий зсув, дозволяючи відфільтрованій лінії максимально точно повторювати оригінал.
4. **Інтерактивне керування (Sliders & Checkboxes)** Надає користувачу панель для динамічної зміни параметрів у режимі реального часу. Дозволяє миттєво спостерігати, як зміна частоти зрізу впливає на якість згладжування.
5. **Оптимізація обчислень (State Management)** Забезпечує статичність шумового патерну: шум генерується наново лише при зміні його власних параметрів. Зміна характеристик гармоніки не призводить до хаотичної зміни "малюнка" перешкод.
